# Stream (Online) ML Team

This is the folder for the **Stream (Online) Model Development**.

## Instructions for Collaboration


1. Create and checkout a new task branch (`task/<task-name>`) from `online-detection`

    > For example, if the task was to load the data: `task/load-dataset`

2. Install required packages:

    ```bash
    pip install -r online-detection/requirements.txt
    ```

3. Create a new `.py` or `.ipynb` file under the `online-detection/tasks` folder, that contains your work.

    > This file can explore the task, print or visualize things.

4. Select the main part of your work and add it to `online-detection/main.py`.

    > Only justified additions will be approved in this file, as this serves our main development file.

5. Push your changes to your branch
6. Create a Pull Request on GitHub as `task/<task-name> --> online-detection`
7. Wait for your project leader to approve your request.
